# NSFC 2026年度申报政策速查

> 具体政策以基金委当年指南为准。

---

## 2026年度主要政策变化

### 申请书结构改革（"瘦身提质"）

正文原则上**不超过30页**。

**面上/青C等项目结构**：
```
（一）立项依据与研究内容（建议8000字以下）
  1. 立项依据
  2. 研究内容、研究目标、关键科学问题
  3. 研究方案及可行性分析
  4. 特色与创新之处
  5. 年度研究计划及预期研究结果
（二）研究基础与工作条件
  1. 研究基础
  2. 工作条件
（三）其他需要说明的情况
```

**重大研究计划等项目结构**：
```
（一）重点阐述内容（建议不超过5000字）
  1. 研究内容、必要性及与重大研究计划的关系
  2. 关键科学问题、研究方案及可行性分析
  3. 年度研究计划及预期研究成果
（二）研究基础与工作条件
（三）其他需要说明的情况
```

### 代表作制度变化
- 代表作需**说明个人贡献情况**（2026年新要求）
- 优先选择第一作者
- 代表作要与申请选题紧密相关
- 附件备注中说明每篇文章与课题的关联

### AI使用规范
- ✅ 可用AI辅助跟踪研究动态、收集整理参考文献
- ✅ 必须人工核实AI生成信息的真实性和准确性
- ✅ 必须全面如实声明使用情况并标识
- ❌ **不得使用AI直接生成申请书**
- ❌ **不得使用未经核实的生成内容**
- ⚠️ AI整理文献时会"一本正经地胡说八道"，务必核实

## 项目类型与定位

### 青年科学基金C类（青C）
| 项目 | 说明 |
|------|------|
| **定位** | 人才类项目，重点评价**创新潜力** |
| **年龄** | 男性35岁以下，女性40岁以下 |
| **期限** | 3年 |
| **经费** | 约30万元 |
| **限制** | 同层次人才计划同一年度只能申请一项 |
| **新变化** | 2026年交叉科学部新增受理青C；博士生/优秀本科生也可申请 |

### 面上项目
| 项目 | 说明 |
|------|------|
| **定位** | 全面均衡布局，瞄准科学前沿，促进学科发展，激励原始创新 |
| **期限** | 4年 |
| **经费** | 约50-60万元 |
| **合作** | 允许合作单位不超过2个 |
| **评审** | 分"自由探索类"和"目标导向类"两套评审要点 |

### 重点项目
| 项目 | 说明 |
|------|------|
| **限项** | 2026年调整：牵头+承担合计限一项 |
| **经费** | 约260万元+ |
| **期限** | 4-5年 |

### 重大研究计划
| 项目 | 说明 |
|------|------|
| **特点** | 有指南，需要"打榜" |
| **重点支持** | 4年，约260-400万元 |
| **要求** | 需阐述与计划总体目标的关系 |

### 联合基金
| 项目 | 说明 |
|------|------|
| **特点** | 有指南，需对接甲方需求 |
| **类型** | 区域创新发展联合基金等 |
| **重点** | 4年，约260万元 |
| **关键** | 理解甲方意图，体现协同创新 |

### 其他新变化
- 交叉学部新增工具领域支持（T05代码）
- 新增**青A赛道**（35岁以内）
- 设立**科学工具赛道**
- 试点重大非共识项目遴选

## 评审流程

```
形式审查/初审 → 通讯评审（函评，5-7位专家）→ 会议评审
```

- 评审主体从小同行向**大同行**转变
- 评审时间极端约束（约20分钟/份）
- 不予受理后可申请复审（成功率极低：166申请→11重新送审→2获资助）

---

## 申请代码与学部

- **9个科学部** + 1个国际科研资助部
- 第一申请代码决定送审的处和会评专家
- 关键词匹配函评专家
- **选择学部要考虑**：哪个学部的专家更能欣赏你的工作

---

## 科研诚信底线

**零容忍**：
- ❌ 不得抄袭、剽窃、伪造数据
- ❌ 不得购买/代写申请书
- ❌ 不得打招呼、围会
- ❌ 不得探听评审信息
- ❌ 不得向评审专家提供礼品礼金
- 违规后果严重（通报、取消资格等）

3. 材料不完整
4. 格式严重不规范
5. 信息虚假
6. 图片打不开/乱码
